# Skill-Gap-Analysis
Skill Gap Analysis: Comparing Industry Demand vs Candidate Profles
• Analyzed job postings and candidate resumes to identify gaps between industry-required skills and candidate
skill sets.
• Performed data collection, preprocessing, and skill extraction to compare demand and supply trends across
multiple domains.
• Developed an interactive dashboard for visualizing skill gaps and market demand patterns.
• Tech Stack: Python, Pandas, NumPy, Plotly, Excel, Data Visualization, Data Preprocessing, Streamlit
